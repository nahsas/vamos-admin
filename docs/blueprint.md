# **App Name**: SejadiKopi

## Core Features:

- Dashboard: Display pending, completed, processing, and today's orders.
- Order: Create, read, update, and delete orders.
- History: View a comprehensive history of all orders.
- Report: Generate and view reports on orders and sales.
- Menu Manager: Manage the menu items available for ordering.

## Style Guidelines:

- Primary color: Coffee brown (#6F4E37) for a warm and inviting feel.
- Secondary color: Creamy white (#FFFAF0) to provide a clean and contrasting background.
- Body font: 'Roboto' for readability and a modern aesthetic.
- Headline font: 'Montserrat' for a bold and clear heading style.
- Use coffee-related icons with a minimalist design.
- Maintain a clean and organized layout for easy navigation.
